Balance API

The balance contains information about the amount of money available on your Omise account.

Object Attributes

Name Type Description
object string

The string balance.
livemode boolean

Whether this is a Live (true) or Test (false) balance.
available integer

Available transferable balance in the smallest currency unit (e.g. satang, cent, ... )
total integer

Total balance in the smallest currency unit (e.g. satang, cent, ... )
currency string

The currency as its lower-cased international 3-letter code, defined by the ISO 4217 standard.

Example

  • Default JSON Representation 

    {
  "object": "balance",
  "livemode": false,
  "available": 0,
  "total": 2742290,
  "currency": "thb"
}
