Balance

The balance contains information about the amount of money available on your Omise account

Table of Contents

  • The balance object
  • Retrieving the balance

The Balance Object

Parameter Value
object (string) 'balance'
livemode (boolean) Whether this is a Live (true) or Test (false) balance.
available (integer) Available transferable balance in the smallest currency unit (e.g. satang, cent, ... )
total (integer) Total balance in the smallest currency unit (e.g. satang, cent, ... )
currency (string) The currency as its lower-cased international 3-letter code, defined by the ISO 4217 standard.

Retrieving the Balance

Endpoint

GET https://api.omise.co/balance

Request parameters

None.

Curl

curl https://api.omise.co/balance \
  -u skey_test_4xs8breq3htbkj03d2x:

Ruby

balance = Omise::Balance.retrieve

# After retrieving it, the balance can be refreshed:
balance.reload

Python

balance = omise.Balance.retrieve()

# After retrieving it, the balance can be refreshed:
balance.reload()

C#

var balance = client.BalanceService.GetBalance();

PHP

$balance = OmiseBalance::retrieve();

# After retrieving it, the balance can be refreshed:
$balance->reload();

JSON Response

{
  "object": "balance",
  "livemode": false,
  "available": 0,
  "total": 0,
  "currency": "thb"
}
