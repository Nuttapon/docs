The Transfer Object

Parameter Value
object (string) 'transfer'
id (string) Transfer Id
amount (integer) Transfer amount in the smallest currency unit (e.g. satang, cent, ... )
currency (string) The currency as its lower-cased international 3-letter code, defined by the ISO 4217 standard.
created (datetime, format: iso8601) Creation date of the transfer

List all transfers

Endpoint

GET https://api.omise.co/transfers

Request Parameters

None

Response parameters

A paginated list of transfers

Curl

curl https://api.omise.co/transfers -u skey_test_4xsjvwfnvb2g0l81sjz:

Ruby

transfers = Omise::Transfer.retrieve

C#

var transfers = client.TransferService.GetAllTransfers();

JSON Response

{
    "object": "list",

    "limit": 20,
    "total": 1,
    "data": [
        {
            "object": "transfer",
            "id": "trsf_test_4xs5px8c36dsanuwztf",
            "amount": 100000,
            "currency": "thb",
            "created": "2014-10-14T05:19:44Z"
        }
    ]
}

Create a transfer

Endpoint

POST https://api.omise.co/transfers

Request Parameters

Parameter Value
amount (optional) Transfer amount in the smallest currency unit (e.g. satang, cent, ... )

Curl

curl https://api.omise.co/transfers \
  -X POST \
  -u skey_test_4xsjvwfnvb2g0l81sjz: \
  -d "amount=100000" \

Ruby

transfer = Omise::Transfer.create({
  amount: 100000
})

C#

var transfer = client.TransferService.CreateTransfer(100000);

JSON Response

{
  "object": "transfer",
  "id": "trsf_test_4xs5px8c36dsanuwztf",
  "amount": 100000,
  "currency": "thb",
  "created": "2014-10-14T05:19:44Z"
}

Retrieve a transfer

Endpoint

GET https://api.omise.co/transfers/TRANSFER_ID

Request Parameters

None

Curl

curl https://api.omise.co/transfers/trsf_test_4xs5px8c36dsanuwztf \
  -X GET \
  -u skey_test_4xsjvwfnvb2g0l81sjz:

Ruby

transfer = Omise::Transfer.retrieve("trsf_test_4xs5px8c36dsanuwztf")

C#

var transfer = client.TransferService.GetTransfer("trsf_test_4xs5px8c36dsanuwztf");

JSON Response

{
  "object": "transfer",
  "id": "trsf_test_4xs5px8c36dsanuwztf",
  "amount": 100000,
  "currency": "thb",
  "created": "2014-10-14T05:19:44Z"
}
