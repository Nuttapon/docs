The Transfer Object

Parameter Value
object (string) 'transfer'
id (string) Transfer Id
amount (integer) Transfer amount in the smallest currency unit (e.g. satang, cent, ... )
currency (string) The currency as its lower-cased international 3-letter code, defined by the ISO 4217 standard.
created (datetime, format: iso8601) Creation date of the transfer

List all transfers

Endpoint

GET https://api.omise.co/transfers

Request Parameters

None

Create a transfer

Endpoint

POST https://api.omise.co/transfers

Request Parameters

Parameter Value
amount (optional) Transfer amount in the smallest currency unit (e.g. satang, cent, ... ). Defaul to the amount in your available balance.

Curl

curl https://api.omise.co/transfers \
  -X POST \
  -u skey_test_4xsjvwfnvb2g0l81sjz: \
  -d "amount=100000" \

Ruby

transfer = Omise::Transfer.create({
  amount: 100000
})

Python

transfer = omise.Transfer.create(amount=100000)

C#

var transfer = client.TransferService.CreateTransfer(100000);

PHP

$transfer = OmiseTransfer::create(array(
    'amount' => 100000
));

JSON Response

Update a transfer

Endpoint

PATCH https://api.omise.co/transfers/TRANSFER_ID

Request Parameters

Parameter Value
amount (optional) Transfer amount in the smallest currency unit (e.g. satang, cent, ... ). Defaul to the amount in your available balance.

Curl

curl https://api.omise.co/transfers/trsf_test_4y3miv1nhy0dceit4w4 \
  -X PATCH \
  -u skey_test_4xsjvwfnvb2g0l81sjz: \
  -d "amount=50000" \

Ruby

transfer = Omise::Transfer.retrieve("trsf_test_4y3miv1nhy0dceit4w4")
transfer.amount = 50000
transfer.save

Python

transfer = omise.Transfer.retrieve("trsf_test_4y3miv1nhy0dceit4w4")
transfer.update(amount=50000)

# Or alternatively:
transfer.amount = 50000
transfer.update()

C#

var transfer = client.TransferService.UpdateTransfer("trsf_test_4y3miv1nhy0dceit4w4", 50000);

PHP

$transfer = OmiseTransfer::retrieve('trsf_test_4y3miv1nhy0dceit4w4');
$transfer['amount'] = 50000;
$transfer->save();

JSON Response

{
  "object": "transfer",

Destroy a transfer

Endpoint

DELETE https://api.omise.co/transfers/TRANSFER_ID

CURL

curl https://api.omise.co/transfers/trsf_test_4y3miv1nhy0rceit4w4 \
  -X DELETE \
  -u skey_test_4xsjvwfnvb2g0l81sjz:

Ruby

transfer = Omise::Transfer.retrieve("trsf_test_4y3miv1nhy0rceit4w4")
transfer.destroy
transfer.destroyed? # => true

Python

transfer = omise.Transfer.retrieve("trsf_test_4y3miv1nhy0rceit4w4")
transfer.destroy()
transfer.destroyed # => true

C#

var deleteResult = client.TransferService.DeleteTransfer("trsf_test_4xs5px8c36dsanuwztf");
// deleteResult.deleted => true

PHP

$transfer = OmiseTransfer::retrieve('trsf_test_4y3miv1nhy0dceit4w4');
$transfer->destroy();
$transfer->isDestroyed(); # => true

JSON Response

{
  "object": "transfer",
  "id": "trsf_test_4y3miv1nhy0rceit4w4",
  "livemode": false,
  "deleted": true
}
