Account API

Object Attributes

Name Type Description
object string

The string account.
id object_id

The ACCOUNT_ID.
email string

Registration email.
created datetime

Creation date of the account in ISO 8601 format.

Example

  • Default JSON Representation 

    {
  "object": "account",
  "id": "acct_4xs8bre8a8vhrgijcjg",
  "email": null,
  "created": "2014-10-20T08:21:42Z"
}

Retrieve the account

- GET https://api.omise.co/account

It may be interesting to note that both Test and Live keys will return the same account.
