The Account Object

Parameter Value
object (string) 'account'
id (string) Account ID
email (string) Registration email
created (datetime, format: iso8601) Creation date of the account

Retrieving the Account

Endpoint

GET https://api.omise.co/account
It may be interesting to note that both Test and Live keys will return the same account.

Request parameters

None.

Curl

curl https://api.omise.co/account \
  -u skey_test_4xs8breq3htbkj03d2x:

Ruby

account = Omise::Account.retrieve

# After retrieving it, the account can be refreshed:
account.reload

C#

var account = client.AccountService.GetAccount();

JSON Response

{
  "object": "account",
  "id": "acct_4xs8bre8a8vhrgijcjg",
  "email": null,
  "created": "2014-10-20T08:21:42Z"
}
