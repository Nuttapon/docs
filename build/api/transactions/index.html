The Transaction Object

Parameter Value
object (string) 'transaction'
id (string) Transaction Id
type (string) 'credit' or 'debit'
amount (integer) Transaction amount in the smallest currency unit (e.g. satang, cent, ... )
currency (string) The currency as its lower-cased international 3-letter code, defined by the ISO 4217 standard.
created (datetime, format: iso8601) Creation date of the transfer

List all transactions

Endpoint

GET https://api.omise.co/transactions

Request Parameters

None parameters

Response

A paginated list of transactions

curl https://api.omise.co/transactions -u skey_test_4xsjvwfnvb2g0l81sjz:

transactions = Omise::Transaction.retrieve

var transactions = client.TransactionService.GetAllTransactions();

Retrieve a transaction

Endpoint

GET https://api.omise.co/transactions/TRANSACTION_ID

Request Parameters

None

Curl

curl https://api.omise.co/transactions/trxn_test_4xuy2z4w5vmvq4x5pfs \
  -X GET \
  -u skey_test_4xsjvwfnvb2g0l81sjz:

Ruby

transaction = Omise::Transaction.retrieve("trxn_test_4xuy2z4w5vmvq4x5pfs")

C#

var transaction = client.TransactionService.GetTransaction("trxn_test_4xuy2z4w5vmvq4x5pfs");
