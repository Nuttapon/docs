Omise Development Libraries

We offer multiple libraries for development in major programming languages.

Officially supported libraries

  • PHP Supports PHP versions 5.3+
  • Ruby Omise RubyGem supports Ruby 2.0+
  • Python Supports Python 2.6 <-> 3.3
  • NodeJS Tested with Node version 0.10.32+
  • Java Supports Java 1.6 and up.
  • .Net Supports .Net framework 2.0, 3.5, 4.0 or 4.5
  • Android SDK Supports 2.2 (API Level 8) or above
  • iOS SDK For iOS development with examples
  • iOS Swift Library for Swift 2.0

Community libraries

  • Go Omise Go Library

Contributing

Our libraries are opensource and available in Github.

If you would like to contribute by adding features that you would like, or by fixing issues or bugs, please send a pull request on Github. We love pull requests from everyone.

How to contribute to opensource projects in Github: https://guides.github.com/activities/contributing-to-open-source/
